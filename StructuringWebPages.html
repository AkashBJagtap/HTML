Every web page has only 1 main element

Each section element should have heading

Article can have different section

How we use semantics elemets really depends on what you have on the web pages

Main element is used to reoresent main content of the page

We use section element to group related content

We use header element to represent introductory content which can belong to a page, section, or an article. Same for footer element. Side Element is used to represent content not directly related to the main content
